## Resumo
Resolução do primeiro TPC. Este primeiro trabalho consistiu na análise de um dataset de uma oficina automóvel e das intervenções feitas, e na definição de um website para explorar esse dataset. 

---

## Lista de resultados
* **TPC1.py**: Script principal.
* **dataset_reparacoes.json**: Ficheiro fonte com os dados.
* **index.html**: Página principal.
* **lista_reparacoes.html**: Índice completo de todas as intervenções realizadas.
* **lista_intervencoes.html**: Índice alfabético dos códigos e tipos de serviço.
* **lista_modelos.html**: Índice de marcas e modelos que passaram pela oficina.
* **reparacoes/*.html**: Páginas individuais com o detalhe de cada reparação.
* **intervencoes/*.html**: Páginas de detalhe para cada tipo de intervenção.

* **modelos/*.html**: Páginas por marca/modelo.

## Execução
Para executar, basta correr o comando "python TPC1.py". É necessário vereficar que o ficheiro com o dataset está presente na pasta. É possível ver o resultado ao abrir o ficheiro index.html na pasta output gerada
